# [senator](https://github.com/senatorv2/senatorv4)
*TG-CLI based broadcasting bot!*
# Install
```
cd $HOME
git clone https://github.com/senatorv2/senatorv4.git
cd senatorv4
chmod +x install.sh
chmod +x launch.sh
chmod +x tg
./install.sh
./launch.sh
# Enter a phone number & confirmation code.
```
Create a bot!
